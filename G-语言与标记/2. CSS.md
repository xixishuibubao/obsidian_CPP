# 1. CSS 样式表

CSS（Cascading Style Sheets）控制 HTML/XML 文档的布局与视觉呈现。本 vault 以 Markdown 为主；CSS 在发布网页、Obsidian 主题 snippet、文档站点时用到。

## 【核心精简提纲】

### 记忆主干

**选择器 → 声明块 `{ 属性: 值; }`**；层叠（Cascade）决定冲突时谁生效；布局核心：**盒模型** + **Flex**（一维）/ **Grid**（二维）。

### 核心要点清单

- 选择器：元素 `p`、类 `.card`、ID `#header`、后代 `article pre`
- 盒模型：`content + padding + border + margin`；`box-sizing: border-box` 让 `width` 含 padding/border
- Flex：`display: flex` + `gap` / `align-items` / `justify-content`
- Obsidian：主题 `.obsidian/themes/`，片段 `.obsidian/snippets/*.css`
- 笔记正文不写 `<style>`，保持纯 MD 可发布

### 关键流程

```
HTML 结构 → 链接 CSS → 选择器匹配 → 层叠算最终样式 → 渲染
Obsidian：启用 snippet → 重载 → 检查阅读视图
```

## 【完整拓展详情】

## 1.1 常用场景

| 场景 | 说明 |
|------|------|
| 网页 / 文档站 | 配合 HTML、Vite/Hugo 等静态站点 |
| Obsidian | `.obsidian/snippets/*.css` 微调阅读样式 |
| README / 门户 | GitHub 不支持自定义 CSS；本地 HTML 预览需要 |
| 与 Markdown | MD 管结构，CSS 管外观 |

### 适用边界

- **适合**：需要统一视觉、响应式布局、主题切换
- **不适合**：GitHub 原生 README 渲染；纯 MD 发布流水线（应外置样式表）

## 1.2 基本结构与选择器

```css
/* 选择器 { 属性: 值; } */
body {
    font-family: system-ui, sans-serif;
    line-height: 1.6;
    max-width: 720px;
    margin: 0 auto;
}

.note-title {
    color: #00599c;
    border-bottom: 1px solid #ddd;
}

#main pre {
    background: #f6f8fa;
    padding: 12px;
    overflow-x: auto;
}
```

| 选择器 | 示例 | 含义 |
|--------|------|------|
| 元素 | `p { }` | 所有 `<p>` |
| 类 | `.card { }` | `class="card"` |
| ID | `#header { }` | `id="header"`（页内应唯一） |
| 后代 | `article pre { }` | article 内任意层级 pre |
| 子代 | `ul > li { }` | 直接子 li |
| 伪类 | `a:hover { }` | 交互状态 |

### 层叠（Cascade）简述

同一元素多条规则冲突时，大致优先级：**!important > 内联 style > ID > 类/属性/伪类 > 元素**；同级后写覆盖先写。详情可查 MDN *Specificity*。

### 典型错误

- 类名拼写与 HTML `class` 不一致 → 样式不生效（用 DevTools 检查是否匹配）
- 忘记单位：`width: 720` 无效，需 `720px` 或 `%`
- 全局 `*` 重置不当导致 Obsidian 内置样式被破坏

## 1.3 盒模型与布局

```css
.box {
    box-sizing: border-box;
    width: 100%;
    padding: 1rem;
    margin: 0.5rem 0;
    border: 1px solid #ccc;
}

.flex-row {
    display: flex;
    gap: 1rem;
    align-items: center;
    justify-content: space-between;
}

.grid-cards {
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(240px, 1fr));
    gap: 1rem;
}
```

| 模型 | 用途 |
|------|------|
| 盒模型 | 控制尺寸与间距；`border-box` 避免 padding 撑破宽度 |
| Flexbox | 导航栏、工具条、单行对齐 |
| Grid | 卡片墙、表单多列 |

### 方案对比

| 需求 | 推荐 |
|------|------|
| 单行工具栏 | Flex |
| 二维卡片网格 | Grid |
| 经典居中（单元素） | Flex `align/justify-center` 或 `margin: auto`（块级定宽） |

## 1.4 与 Obsidian 实践

- 主题：`.obsidian/themes/`（本 vault git 白名单同步）
- 自定义片段：`.obsidian/snippets/my.css`，设置 → 外观 → CSS 片段中启用
- 常用变量：社区主题常暴露 `--accent-h` 等 CSS 变量，snippet 可覆盖
- **勿在笔记正文写 `<style>`**：不利于纯 MD 发布与多端同步

### 实操案例：代码块可读性

```css
/* .obsidian/snippets/code-readable.css */
.markdown-preview-view pre {
    line-height: 1.45;
    font-size: 0.9em;
}
```
