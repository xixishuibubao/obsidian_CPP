# 1. CSS 样式表

CSS（Cascading Style Sheets）控制 HTML/XML 文档的 **布局与视觉呈现**。本 vault 以 Markdown 为主；CSS 在发布网页、Obsidian 主题 snippet、文档站点时用到。

## 1.1 常用场景

| 场景 | 说明 |
|------|------|
| **网页 / 文档站** | 配合 HTML、静态站点生成器 |
| **Obsidian** | `.obsidian/snippets/*.css` 微调阅读样式 |
| **README / 门户** | GitHub 不支持自定义 CSS，但本地预览 HTML 时需要 |
| **与 Markdown** | MD 管结构，CSS 管外观；见 [[G-语言与标记/1. Markdown|Markdown]] |

## 1.2 基本结构

```css
/* 选择器 { 属性: 值; } */
body {
    font-family: system-ui, sans-serif;
    line-height: 1.6;
    max-width: 720px;
    margin: 0 auto;
}

.note-title {
    color: #00599c;
    border-bottom: 1px solid #ddd;
}

#main pre {
    background: #f6f8fa;
    padding: 12px;
    overflow-x: auto;
}
```

| 选择器 | 示例 | 含义 |
|--------|------|------|
| 元素 | `p { }` | 所有 `<p>` |
| 类 | `.card { }` | `class="card"` |
| ID | `#header { }` | `id="header"` |
| 后代 | `article pre { }` | article 内 pre |

## 1.3 盒模型与布局（速查）

```css
.box {
    box-sizing: border-box;
    width: 100%;
    padding: 1rem;
    margin: 0.5rem 0;
    border: 1px solid #ccc;
}

.flex-row {
    display: flex;
    gap: 1rem;
    align-items: center;
}
```

- **盒模型**：content + padding + border + margin
- **Flexbox**：一维排列（导航栏、工具条）
- **Grid**：二维网格（卡片墙）

## 1.4 与 Obsidian

- 主题在 `.obsidian/themes/`（git 白名单同步）
- 自定义片段：`.obsidian/snippets/my.css`，在设置中启用
- 勿在笔记正文写 `<style>`（不利于纯 MD 发布）
