# 1. 编程常用英语词汇

阅读源码、文档、API 时的术语速查；按主题分组，遇生词先猜再查。

## 【核心精简提纲】

### 记忆主干

编程英语 ≈ 小词高频：`config` `status` `param` `callback` `register`；系统词：`kernel` `mutex` `scheduler`；OOP：`Encapsulation` `Inheritance` `Polymorphism`。

### 核心要点清单

- **通用**：iterator, callback, register, version, config, status, param, event
- **系统/嵌入式**：kernel, IRQ, MMU, mount, driver, device, sector, scheduler
- **并发**：mutex, Semaphore, critical sections, race condition, woken
- **OOP**：Encapsulation, Inheritance, Polymorphism, global, element
- **构建**：build, manifest, binary, path, cache, format
- **天文/项目**：Telescope, preview, theodolite, focus

### 学习路径

1. 读代码时标记陌生标识符，回查本表
2. 按模块积累（网络/驱动/构建各一套）
3. 与 [[G-语言与标记/1. Markdown|Markdown]] 双语笔记习惯结合：英文术语 + 中文解释

### 关键结论 / 流程

```plain
遇术语 → 本表速查 → 记入个人词单 → 写笔记时保留英文原词
```
