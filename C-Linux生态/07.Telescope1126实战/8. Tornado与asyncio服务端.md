# 1. Tornado 与 asyncio 服务端（Telescope1126 案例）

**Telescope1126Server** 用 **Tornado 6 + asyncio** 做 HTTP/WebSocket 服务，子进程 Executor 同样基于 asyncio 事件循环。

前置：[[2. Server双进程架构]] · 关联：[[D-系统与架构/03.并发与内存/3. 线程与异步|线程与异步]] · [[C-Linux生态/05.网络编程/2. IO模型与muduo设计|IO 模型]]（Reactor 思想相通）

## 1.1 为什么不是 Flask / Django

| 框架 | 模型 | 本项目 |
|------|------|--------|
| Flask | 同步 WSGI，一线程/进程一请求 | 不适合长连接 WS + 高并发 notify |
| Django | 全栈 ORM，偏重 | 板端嵌入式不需要 |
| **Tornado** | 单线程 asyncio + 非阻塞 I/O | **采用** — HTTP + WebSocket + IPC 同循环 |

Tornado 与 Muduo 同属 **Reactor**：事件循环等待 fd 就绪，回调处理读写。

## 1.2 程序入口模式

```python
# HoriVisionDriverServer.py
async def main():
    StartUpScripts(HoriVisionDevices)
    app = HorivisionAppServer(HoriVisionDevices)
    await app.startup()                    # IpcServer + ExecutorSupervisor
    http_server = app.listen(8999)
    await shutdown_event.wait()
    http_server.stop()
    await app.shutdown()

asyncio.run(main())
```

要点：

- **`async def main()`** — 整个服务跑在一个 asyncio 事件循环里
- **`app.listen(8999)`** — 非阻塞监听，不阻塞 startup 里的 IPC
- **优雅退出** — `shutdown_event` + `SIGINT/SIGTERM`（Linux）

## 1.3 Application 与 Handler

```python
class HorivisionAppServer(tornado.web.Application):
    def __init__(self, horivision_devices):
        handlers = [
            (r"/executor/task/start/", TaskStartHandler),
            (r"/ws/device/", APIWebsocketHandler),
            # ...
        ]
        super().__init__(handlers, debug=True)
```

| 类型 | 基类 | 用途 |
|------|------|------|
| HTTP API | `tornado.web.RequestHandler` | GET/POST JSON |
| WebSocket | `tornado.websocket.WebSocketHandler` | 设备实时控制 |
| 自定义 | `SharedParameter.BaseHandler` | 项目统一 JSON 响应 |

Handler 可以是 **`async def get/post`**，内部 `await` 异步操作。

## 1.4 asyncio 在本项目的三处用法

### 1.4.1 IPC：Unix Stream + StreamReader/Writer

```python
# ServerState.py — IpcServer
self._server = await asyncio.start_unix_server(
    self._handle_connection, path=IPC_SOCK_PATH
)

async def send_cmd(self, task_type, params):
    fut = asyncio.get_event_loop().create_future()
    self._pending_cmds[cmd_id] = (time.time(), fut)
    self._writer.write(make_msg(CMD, ...))
    await self._writer.drain()
    return await asyncio.wait_for(fut, timeout=5.0)
```

模式：**写 socket → 注册 Future → 收到 cmd_resp 时 set_result**。

### 1.4.2 Executor：TaskRunner + asyncio.Queue

```python
# executor.py — 串行任务队列
self._queue: asyncio.Queue = asyncio.Queue()

async def _worker(self):
    while True:
        item = await self._queue.get()
        await self._run_one_task(item)   # 一次只跑一个
```

硬件互斥 → **单 worker 协程**，不是线程池。

### 1.4.3 阻塞驱动调用：`run_sync`

相机 `.so` 方法是**同步阻塞**的，不能在协程里直接长时间占用循环：

```python
# SingleExposure.py
await self.ctx.run_sync(this_camera_driver.ask_camera_exposure, raw_path, seconds)
notify_msg = await self.ctx.wait_for_device_event(device_id=..., timeout=...)
```

`run_sync` 通常把阻塞函数丢到 **executor 线程池**，避免卡住 IPC/HTTP。

## 1.5 WebSocket 广播：WsHub

```python
class WsHub:
    def broadcast(self, payload: dict):
        msg = json.dumps(payload)
        for c in self._clients:
            c.write_message(msg)
```

Executor 通过 IPC 上报 `task_finished` → `IpcServer` 更新 `StateStore` → **`hub.broadcast(...)`** 推给所有 App 客户端。

## 1.6 PeriodicCallback 与看门狗

```python
# 每 5 秒检查 Executor 心跳
tornado.ioloop.PeriodicCallback(self._hb_watchdog, 5000).start()

def _hb_watchdog(self):
    if (time.time() - last) > HB_TIMEOUT:   # 15 秒
        self.store.executor_online = False
        self.hub.broadcast({"type": "executor_status", "online": False})
```

也可用 **`asyncio.create_task` + `asyncio.sleep`** 实现同等逻辑。

## 1.7 与 Muduo / epoll 概念对照

| Muduo / Linux | Tornado / asyncio |
|---------------|-------------------|
| `EventLoop::loop()` | `asyncio.run()` / `IOLoop.current()` |
| `Channel` + fd | `add_reader` / `StreamReader` |
| `TcpConnection` | `RequestHandler` / `WebSocketHandler` |
| 线程池处理计算 | `run_in_executor` / `ctx.run_sync` |

## 1.9 常见坑

| 坑 | 说明 |
|----|------|
| 在 Handler 里 `time.sleep(10)` | 阻塞整个事件循环，8999 全站卡死 |
| Windows 开发 IPC | UDS 不可用，只能读代码 |
| `debug=True` | 生产板端应关 debug，避免泄露栈 |
| 混用线程与 asyncio | 改共享状态要加锁；优先 Queue 传递消息 |
