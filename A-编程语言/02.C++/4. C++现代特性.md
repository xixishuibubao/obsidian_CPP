# 4. C++ 现代特性

C++11/14/17/20 核心现代特性速查：右值引用与移动语义、智能指针、lambda、`enum class`、`noexcept` 等。

## 【核心精简提纲】

### 记忆主干

**移动语义避免拷贝 · 智能指针 RAII 管理 · lambda 就地闭包 · `enum class` 类型安全 · `noexcept` 优化移动 · `=default/=delete` 显式控制**

### 核心要点清单

- **右值引用 `&&`**：绑定到临时对象，实现资源转移而非拷贝
- **移动构造/赋值**：应标记 `noexcept` 以启用容器优化（如 `vector` 扩容时 move 而非 copy）
- **引用叠加**：有 `&` 则 `&`，全 `&&` 则 `&&`
- **`std::move`**：本质是 `static_cast<T&&>`，只转换不移动
- **`std::forward`**：完美转发，保持值类别（左值/右值）不变
- **智能指针**：`unique_ptr`（独占）、`shared_ptr`（共享计数）、`weak_ptr`（弱引用破环）
- **lambda**：`[捕获](参数) -> 返回类型 { 函数体 }`；C++14 支持泛型 lambda
- **`enum class`**：强作用域、可指定底层类型、需强制转换
- **`noexcept`**：声明不抛异常，配合移动语义优化容器性能
- **`=default` / `=delete`**：显式生成或禁用特殊成员函数

## 1. 右值引用与移动语义

### 1.1 基本概念

- **左值**：可以取地址、有持久状态的对象
- **右值**：临时对象、字面量，即将被销毁
- **右值引用 `&&`**：绑定右值，实现资源转移

```cpp
int a = 42;
int& lref = a;       // 左值引用
int&& rref = 42;     // 右值引用，绑定到临时对象
```

### 1.2 移动构造与移动赋值

```cpp
class Buffer {
    char* data_;
public:
    // 移动构造
    Buffer(Buffer&& other) noexcept
        : data_(other.data_) {
        other.data_ = nullptr;    // 置空源对象
    }

    // 移动赋值
    Buffer& operator=(Buffer&& other) noexcept {
        if (this != &other) {
            delete[] data_;
            data_ = other.data_;
            other.data_ = nullptr;
        }
        return *this;
    }
};
```

> 移动操作标记 `noexcept` 非常重要：`std::vector` 扩容时，若移动构造为 `noexcept` 则使用移动，否则回退到拷贝。

### 1.3 std::move 与 std::forward

```cpp
// std::move：无条件转为右值引用（仅为转换，不移动）
std::string str = "hello";
std::string dest = std::move(str);  // str 被置空

// std::forward：完美转发，保持原始值类别
template<typename T>
void wrapper(T&& arg) {
    func(std::forward<T>(arg));  // 保持左值/右值
}
```

### 1.4 引用叠加原则

当对一个对象双层引用：有 `&` 则 `&`，全 `&&` 则 `&&`。

| 原始类型 | 叠加 `&` | 叠加 `&&` |
|---------|---------|----------|
| `T&`    | `T&`    | `T&`     |
| `T&&`   | `T&`    | `T&&`    |

## 2. 智能指针

### 2.1 unique_ptr（独占所有权）

```cpp
#include <memory>
auto ptr = std::make_unique<int>(42);
// 不可拷贝，只能移动
auto ptr2 = std::move(ptr);
// 自定义删除器
auto file = std::unique_ptr<FILE, decltype(&fclose)>(
    fopen("test.txt", "r"), &fclose);
```

### 2.2 shared_ptr（共享所有权）

```cpp
auto sptr = std::make_shared<int>(42);
{
    auto sptr2 = sptr;  // 引用计数 +1
}   // 离开作用域，引用计数 -1
// 引用计数归零时自动释放资源
```

### 2.3 weak_ptr（弱引用，破环）

```cpp
auto sptr = std::make_shared<int>(42);
std::weak_ptr<int> wptr = sptr;  // 不增加引用计数
// 使用前需 lock() 提升为 shared_ptr
if (auto locked = wptr.lock()) {
    // 使用 locked
}
```

### 2.4 注意事项

- `shared_ptr` 控制块与对象分别分配，`make_shared` 可合并分配提升性能
- 循环引用时使用 `weak_ptr` 打破
- `unique_ptr` 零开销，优先使用

## 3. lambda 表达式

### 3.1 基本语法

```cpp
[捕获列表](参数列表) -> 返回类型 { 函数体 }

auto add = [](int a, int b) -> int { return a + b; };
auto print = [](const auto& x) { cout << x; };  // C++14 泛型 lambda
```

### 3.2 捕获方式

| 捕获 | 含义 |
|------|------|
| `[]` | 不捕获任何外部变量 |
| `[=]` | 按值捕获所有外部变量 |
| `[&]` | 按引用捕获所有外部变量 |
| `[this]` | 捕获当前类指针，可访问成员 |
| `[x, &y]` | x 按值，y 按引用 |

```cpp
int x = 1, y = 2;
auto f = [x, &y] { return x + y; };  // x 拷贝，y 引用
// C++14 支持初始化捕获（移动捕获）
auto uptr = std::make_unique<int>(42);
auto g = [u = std::move(uptr)] { return *u; };
```

## 4. enum class

C++11 引入的强类型枚举：

```cpp
enum class Color : uint8_t {
    Red,
    Green,
    Blue
};
Color c = Color::Red;
// 需强制转换
int val = static_cast<int>(c);
```

- 强作用域（不会造成同名冲突）
- 可指定底层数据类型
- 需强制类型转换才能转为整数

## 5. noexcept / =default / =delete

### 5.1 noexcept

声明函数不抛异常，常用于移动构造/赋值以启用容器优化。

```cpp
class A {
public:
    A(A&& other) noexcept;
    A& operator=(A&& other) noexcept;
};
```

### 5.2 =default / =delete

```cpp
class A {
public:
    A() = default;             // 显式生成默认构造
    A(const A&) = delete;      // 禁止拷贝构造
    A& operator=(const A&) = delete;  // 禁止拷贝赋值
};
```
