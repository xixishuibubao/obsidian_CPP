# 1. 管道

## 1.1 无名管道（只能用于父子或兄弟进程）

```c
int pipe_fd[2];  // [0] 读端, [1] 写端
pipe(pipe_fd);
```

## 1.2 有名管道（任意进程通信）

```c
mkfifo("fifo_name", mode);
int fd = open("fifo_name", O_RDONLY);  // 或 O_WRONLY
```

# 2. 文件描述符操作

```c
fd2 = dup(fd);                // 重定向文件符号
fd2 = dup2(old_fd, new_fd);   // 重定向文件内容
int new_fd = fcntl(old_fd, F_DUPFD, fd_no);  // 等同于 dup()
```

# 3. 内存映射（mmap）

```c
#include <sys/mman.h>

void* p = mmap(NULL, length, prot, flags, fd, offset);
// prot: PROT_READ | PROT_WRITE
// flags: MAP_SHARED / MAP_PRIVATE
munmap(p, length);
```

- mmap 的读写权限需 <= 打开 fd 时的权限
- 文件必须有内容且长度 > 0，否则会出错
- 打开文件并映射后即可关闭 fd

扩展文件大小：

```c
lseek(fd, offset, SEEK_END);
write(fd, "0", 1);

// 或
truncate("filename", size);
ftruncate(fd, size);
```

对 mmap 共享写入后，用 `msync` 同步文件：

```c
int msync(void *addr, size_t length, int flags);
// flags: MS_ASYNC（异步）, MS_SYNC（同步）, MS_INVALIDATE（使缓存失效）
```
