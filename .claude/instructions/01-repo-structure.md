# 仓库结构说明

## 目录总览（A-E 语义分组）

仓库使用 `字母前缀-主题/序号.子主题/` 的语义分组结构，共五大模块：

### A-编程语言

| 子目录 | 笔记数 | 内容 |
|--------|--------|------|
| `01.C语言` | 3 | C 基础 · 底层原理 · 系统编程 |
| `02.C++` | 4 | C++ 基础 · 代码规范 · 项目结构 · 模板 |
| `03.Golang` | 1 | Go 语言基础 |

### B-构建与脚本

| 子目录 | 笔记数 | 内容 |
|--------|--------|------|
| `01.构建工具` | 6 | g++ · Makefile · CMake（入门/进阶/复杂项目）· 编译脚本设计 |
| `02.脚本语言` | 3 | Shell · Lua · Python |

### C-Linux生态（最大模块，6个子目录）

| 子目录 | 笔记数 | 内容 |
|--------|--------|------|
| `01.Linux环境` | 6 | vim · 常用指令 · tar · ssh · 动态库与静态库 · rpath |
| `02.Linux系统编程` | 5 | 进程管理 · CPU亲和性 · 用户态/内核态 · 字符设备驱动 · 内核模块开发 |
| `03.Linux开发` | 3 | RK3588 · Yocto · 交叉编译优化选项 |
| `04.调试与优化` | **8**（最大子目录） | gdb · coredump · perf · 文件IO · 分支优化 · 综合调试 |
| `05.网络编程` | 2 | Muduo 网络库 · IO 模型 |
| `06.开源项目分析` | 6 | Muduo/LevelDB/Redis/Nginx 等开源项目分析 |

### D-系统与架构

| 子目录 | 笔记数 | 内容 |
|--------|--------|------|
| `01.架构体系` | 1 | ARM vs X86 汇编 |
| `02.软件架构设计` | 4 | PIMPL · 结构化设计 · CLI 设计 · Claude Code Skill 方法论 |
| `03.并发与内存` | 3 | GC 算法 · atomic 与内存序 · 线程与异步 |
| `04.MCU嵌入式` | 2 | MCU 开发基础 · ARM32 内存注入（UART） |

### E-杂项

| 子目录 | 笔记数 | 内容 |
|--------|--------|------|
| `01.开发工具` | 4 | MobaXterm · CLion · VS2022 · Qt |
| `02.版本管理` | 2 | Git 操作 · log 规范 |
| `03.数据库` | 0 | 🚧 占位（空目录） |
| `04.语言与标记` | 6 | Markdown · CSS · XML · PlantUML · UML · 常用英语 |
| `05.杂项` | 4 | 读书笔记（More Effective C++）· 加载与劫持 · 插件架构 · 临时杂物 |

### 其他顶层目录

| 目录 | 内容 |
|------|------|
| `picture/` | 图片资源，命名格式 `{章节号}-{序号}.png`（如 `5-1-1.png`） |
| `readme/` | 多语言 README 文件 |
| `.claude/` | Claude Code 辅助配置与指令文件 |

## 总体统计

- **总笔记数**：~52 篇
- **最大模块**：C-Linux生态（30 篇笔记）
- **最大子目录**：C-Linux生态/04.调试与优化（8 篇）
- **空目录**：E-杂项/03.数据库（通过 `.gitkeep` 占位）
