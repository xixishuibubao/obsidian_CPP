# 1.操作基础

## 1.1 命令模式

- vim 使用ESC进入命令模式，且打开文件默认是命令模式
- 在命令模式下按键将会被识别为操作符。

## 1.2 短命令

- i 在光标之前开始编辑
- a 在光标之后开始编辑
- I 光标行首开始编辑
- A 光标行尾开始编辑
- o 在光标下一行创建空行
- O 在光标上一行创建空行
- ^ 光标切到行首(第一个非空字符)
- $ 光标切到行尾
- v 进入行选模式
- ctrl v 进入竖选模式
- 0 跳转到行首(这是数字0)
- w 跳转到上一个单词
- b 跳转到下一个单词
- hjkl分别对应 左移，下移，上移，右移
- d 删除当前
- d+ 数字+d 删除多少行
- c 删除并进入插入模式
- u 撤销到上一步操作
- ctrl r 回退一步撤销
- 数字+ 向后跳转数字行
- 数字- 向前跳转数字行
- gg 文件头
- G 文件尾
- /向后搜索
- ？向前搜索（搜索模式下 按enter确定，然后按n查找下一个，N上一个）

## 1.2 长命令

- ：w 保存
- ：q 退出
- : q!强制退出
- ：wq 保存并退出
- ：set number显示行号
