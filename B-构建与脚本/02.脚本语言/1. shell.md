# 1. Shell 脚本

Bash/Sh 是 Linux 构建与运维的胶水层：`configure`、[[B-构建与脚本/01.构建工具/2. makefile|Makefile]] 命令、CI 脚本、板端部署均依赖 Shell。本笔记为语法速查；构建链见 [[B-构建与脚本/01.构建工具/5. 编译脚本设计|编译脚本设计]]（Nginx `auto/` 探测脚本）。

## 【核心精简提纲】

### 记忆主干

**无类型字符串变量** + `$`/`${}` 展开 + `` `cmd` ``/`$(cmd)` 命令替换 + `[ ]`/`[[ ]]` 测试。脚本首行 `#!/bin/bash`；函数用 `local`；流程 `if/fi`、`case/esac`、`for/while`；文本处理 `sed`/`awk`。

### 核心要点清单

- 变量：`name=value`（无空格）；引用 `${name}`；环境变量 `export`
- 位置参数：`$0` 脚本名、`$#` 个数、`$1`…`$n`、`$?` 上条命令退出码
- 算术：`((n++))`、`let`、`expr`（乘号 `*` 需转义）
- 条件：`[ $a -eq $b ]` 空格敏感；`[[ ... ]]` 支持 `&&`/`||`
- 重定向：`>` 覆盖、`>>` 追加、`2>` 错误、`&>` 全部
- 构建场景：`set -e` 遇错退出、`$CROSS_COMPILE` 传交叉前缀

## 【完整拓展详情】

## 1.1 变量

1. 命名类似 C，定义时**不能有空格**，无类型标识。
2. `string="hello world"` 定义变量；`echo ${string}` 引用需 `$` 与 `{}`。
3. `echo $(pwd)` 命令替换输出当前目录。
4. `#` 注释；`unset` 删除；`readonly` 只读。
5. `#!/bin/bash` 声明解释器（Linux 常用 `/bin/bash`）。
6. `declare -i my_value=42` 声明整型；`echo -e` 解析转义。
7. `" "` 内可有变量与转义；`' '` 内纯字面。
8. 多行注释：`: ' ... '` 块。

9. 环境变量需 `export`：

```bash
export MY_VAR="hello"
MY_VAR="hello"; export MY_VAR
local MY_VAR="hello"   # 函数内局部
readonly MY_VAR
```

## 1.2 变量传参

| 符号 | 含义 |
|------|------|
| `$#` | 参数个数 |
| `$0` | 脚本名 |
| `$1`…`$n` | 第 n 个参数 |
| `$*` | 所有参数合并为单字符串 |
| `$@` | 所有参数列表 `"$1" "$2" …` |
| `$$` | 当前 shell PID |
| `$!` | 后台最后进程 PID |
| `$?` | 上条命令退出码（0=成功） |

`;` 或 `\` 换行作分隔符。

## 1.3 数组

```bash
my_arr=(1 2 3)
declare -A site=(["a"]="typea" ["b"]="typeb")
```

- `arr[@]` / `arr[*]` 全部元素；`!arr[@]` 全部索引；`#arr[@]` 元素个数
- 关联数组需 `declare -A`；Windows 编辑注意 LF 换行

## 1.4 基本运算符

**算术**：`` `expr a + b` `` 或 `((a+b))` 或 `$((a+b))`；`*` 在 expr 中需 `\*`。

**关系**（在 `[ ]` 内，空格敏感）：`-eq` `-ne` `-gt` `-lt` `-ge` `-le`。

**布尔**：`!` `-o`(or) `-a`(and)；`[[ a -lt 10 && b -gt 5 ]]`。

**字符串**：`=` `!=` `-z` `-n`。

**文件测试**：`-f` 普通文件、`-d` 目录、`-e` 存在、`-r` `-w` `-x` 权限等。

**自增**：`let val++`；`((val++))`。

## 1.5 echo 输出

- `-e` 转义；反引号输出字面转义
- `echo text > file` 覆盖；`>>` 追加；`2>` 错误流；`&>` 全部
- `` `linuxcmd` `` 嵌入命令输出

## 1.6 printf

语法同 C，无括号。

## 1.7 通配符

| 符号 | 含义 |
|------|------|
| `*` | 任意长度 |
| `?` | 单字符 |
| `[abc]` / `[a-z]` | 字符类 |
| `{1,2,3}` | 展开多串 |

## 1.8 `${}` 与 `$()`

- `${var}` 变量展开，可嵌套；`${}` 内命令不执行
- `$(cmd)` 命令替换，执行并取输出

## 1.9 test 语句

`test expr` 或 `[ expr ]`；`[[ expr ]]` 支持通配符匹配。

## 1.10 if 语句

```bash
if expr1; then
  cmd1
elif expr2; then
  cmd2
else
  :
fi
```

`then` 后不可加 `;`；空分支用 `:`。

## 1.11 case 语句

```bash
case expr in
  a) cmda ;;
  b) cmdb ;;
  *) cmdx ;;
esac
```

## 1.12 循环

**for 列表**：

```bash
for val in 1 2 3; do echo $val; done
for(( i=1; i<=5; i++ )); do echo $i; done
```

**while 读文件**：

```bash
while read -r line; do
  echo "$line"
done < log.txt
```

**until**：条件为假时循环。

**select**：交互菜单，常配合 `case`。

`break` 跳出循环。

## 1.13 函数

```bash
my_func() {
  local x=1
  return 0
}
my_func arg1 arg2
```

**`${}` 字符串操作**：`${#var}` 长度、`${var#pat}` 删前缀、`${var%pat}` 删后缀、`${var/old/new}` 替换。

**trap**：绑定信号处理。

| 信号 | 说明 |
|------|------|
| 2 SIGINT | Ctrl+C |
| 15 SIGTERM | 默认 kill |
| 0 | 脚本正常退出 |

构建脚本常用：`trap 'rm -f tmp.*' EXIT`。

## 1.14 sed 与 awk

**sed**：`sed 's/old/new/g' file`；`-i` 改原文件；`/` 分隔符可换 `@`。

**awk**：`awk '{print $1}' file`；`-F ','` 指定分隔符；`NR` 行号、`NF` 字段数。

## 1.15 构建脚本惯用法

与 [[B-构建与脚本/01.构建工具/5. 编译脚本设计|编译脚本设计]]、[[C-Linux生态/07.Telescope1126实战/6. 交叉编译与DriverSO部署|DriverSO 部署]] 常见写法：

```bash
set -euo pipefail          # 遇错退出、未定义变量报错
CROSS_COMPILE=${CROSS_COMPILE:-aarch64-linux-gnu-}
CC="${CROSS_COMPILE}gcc"
cmake -B build -DCMAKE_TOOLCHAIN_FILE=toolchain.cmake
cmake --build build -j"$(nproc)"
```
