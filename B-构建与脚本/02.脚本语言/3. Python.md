## 1.1 本项目常用语法速查

### 1.1.1 import 与动态加载

```python
import importlib
from pathlib import Path

# 固定模块
from APIServer.ServerState import StateStore

# 运行时加载 DriverSO 中的 .so（当作 Python 模块）
sys.path.insert(0, "DriverSO")
mod = importlib.import_module("CMOS285CAMERAMain")
cam = mod.CMOS285CAMERAMain("/dev/")
```

### 1.1.3 asyncio 基础

```python
import asyncio

async def worker():
    await asyncio.sleep(1)
    return "done"

async def main():
    task = asyncio.create_task(worker())
    result = await task

asyncio.run(main())
```

Tornado 6 与 asyncio **共用事件循环** — 详见 [[8. Tornado与asyncio服务端]]。

### 1.1.4 ctypes 调 C 库

cfitsio = CDLL("DriverSO/libcfitsio.so")
status = c_int(0)
cfitsio.ffinit(byref(fptr), filename.encode(), byref(status))

见 `SupportLib/fits.py`、[[9. FITS天文图像与后处理]]。ctypes / pybind11 / Embedding 选型见 [[4. C-C++与Python互调用]]。
