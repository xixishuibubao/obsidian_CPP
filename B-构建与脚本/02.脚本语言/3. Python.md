# 1. Python 脚本语言

Python 在本知识库中主要用于 **服务端编排**、Deploy/Test 脚本及 C/C++ 扩展模块调用。驱动层通过 pybind11 暴露给 Python。

## 【核心精简提纲】

### 记忆主干

**项目向速查**：`importlib` 动态加载 DriverSO · `dataclass` 结构化头 · `asyncio` 与 Tornado 共循环 · `ctypes` 调 cfitsio · `subprocess.Popen` 后处理 · `ConfigObj` 读 INI。

### 核心要点清单

- 动态模块：`sys.path` + `importlib.import_module("CMOS285CAMERAMain")`
- 类型：`@dataclass` + `Optional`（见 `AfterProcessDatatype.py`）
- 异步：`async def` 内禁用阻塞 `time.sleep`，用 `await asyncio.sleep`
- C 库：`CDLL("...so")` + `byref(c_int)` 传状态码
- 配置：`ConfigObj("server.ini")` 与 `hardware.ini`
- 与 Shell：HTTP/部署用 Python；交叉编译用 `build_all.sh`

### 自测速答

- `import_module`：运行时按名字加载；`from x import y` 编译期静态绑定
- FITS 用 ctypes：已有 `libcfitsio.so`，无需重写 C 库绑定
- `time.sleep` 在 `async def` 会阻塞整个事件循环

## 1.1 本项目常用语法速查

### 1.1.1 import 与动态加载

```python
import importlib
import sys
from pathlib import Path

from APIServer.ServerState import StateStore

sys.path.insert(0, "DriverSO")
mod = importlib.import_module("CMOS285CAMERAMain")
cam = mod.CMOS285CAMERAMain("/dev/")
```

**原理**：`import_module` 执行模块顶层代码并缓存于 `sys.modules`；DriverSO 目录需在 `sys.path` 中以便找到 `.so` 与同目录 `*.py`。

**踩坑**：路径相对 CWD，部署脚本须 `chdir` 到 Server 根或写绝对路径。

### 1.1.2 dataclass 与类型提示

```python
from dataclasses import dataclass
from typing import Optional

@dataclass
class FitsBasicHeaderData:
    EXPOSURE: float
    GAIN: int
    TELESCOP: Optional[str] = None
```

见 `SharedParameter/AfterProcessDatatype.py`。dataclass 自动生成 `__init__`/`__repr__`，适合 FITS 头等结构化数据。

### 1.1.3 asyncio 基础

```python
import asyncio

async def worker():
    await asyncio.sleep(1)
    return "done"

async def main():
    task = asyncio.create_task(worker())
    result = await task

asyncio.run(main())
```

Tornado 6 与 asyncio **共用事件循环** — 详见 [[D-系统与架构/03.并发与内存/3. 线程与异步|线程与异步]]。

**边界**：CPU 密集或阻塞 I/O 应放 `run_in_executor`，避免卡住 HTTP 与 IPC。

### 1.1.4 ctypes 调 C 库

```python
from ctypes import CDLL, c_int, c_char_p, byref

cfitsio = CDLL("DriverSO/libcfitsio.so")
status = c_int(0)
cfitsio.ffinit(byref(fptr), filename.encode(), byref(status))
```

**选型**：ctypes 适合已有 C ABI、调用面窄；复杂 C++ API 用 pybind11（[[B-构建与脚本/02.脚本语言/4. C-C++与Python互调用|C/C++ 与 Python 互调用]]）。

### 1.1.5 subprocess 后台进程

```python
import subprocess

subprocess.Popen(
    ["/path/raw2jpg", "main_camera", lock, raw, jpg, "1"],
    start_new_session=True,
)
```

见 `SupportLib/AfterExposureProcessPopen.py`。`start_new_session=True` 使子进程脱离终端会话，适合曝光后异步图像处理。
