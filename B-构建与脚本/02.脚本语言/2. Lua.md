# 1. Lua 脚本语言

Lua 是轻量 **嵌入式脚本语言**，单文件解释器、C API 成熟，常用于游戏逻辑、Nginx OpenResty、Redis 脚本、配置 DSL。本知识库主栈以 Python/Shell 为主；Lua 作扩展阅读。

## 1.1 常用场景

| 场景 | 说明 |
|------|------|
| **嵌入 C/C++** | 通过 `lua.h` / `lauxlib.h` 在宿主程序内跑脚本 |
| **OpenResty / Nginx** | 高性能网关逻辑 |
| **游戏 / 工具链** | 热更新逻辑、Wireshark 插件 |
| **Redis** | `EVAL` 原子脚本 |

**本项目**：Telescope1126 以 **Python + pybind11** 为脚本层，未用 Lua；若驱动只需极简 glue 且体积敏感，可评估 Lua embed 替代部分 Python。

## 1.2 基础语法

```lua
-- 变量默认全局；local 局部
local n = 42
local s = "hello"

-- table = 数组 + 字典
local t = { 1, 2, name = "cam" }
print(t[1], t.name)

-- 函数
local function add(a, b)
    return a + b
end

-- 迭代
for i, v in ipairs(t) do
    print(i, v)
end
```

## 1.3 C 嵌入最小示例

```c
#include <lua.h>
#include <lauxlib.h>
#include <lualib.h>

int main(void) {
    lua_State *L = luaL_newstate();
    luaL_openlibs(L);
    if (luaL_dostring(L, "print('hello from lua')") != LUA_OK)
        return 1;
    lua_close(L);
    return 0;
}
```

链接：`-llua`（需安装 `liblua-dev`）。
