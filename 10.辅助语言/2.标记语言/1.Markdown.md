

- Markdown的符号都是英文有效的，输入请注意中英文。
- 回车换行前要加两个空格

# 1.标题

- 使用`#`来表示标题，`#`的数量表示标题的级别，级别数值越大字号越小。
	- ```Markdown
		# 一级标题
		## 二级标题
		### 三级标题
		```

# 2.文本样式

1. *加粗**：使用`**`或`__`包裹文本，例如`**加粗文本**`或`__加粗文本__`都会显示为**加粗文本**。
2. _斜体_：使用`*`或`_`包裹文本，例如`*斜体文本*`或`_斜体文本_`都会显示为*斜体文本*。
3. _**加粗斜体**_：使用`***`或`___`包裹文本，例如`***加粗斜体文本***`或`___加粗斜体文本___`都会显示为_**加粗斜体文本**_。
4. ~~删除线~~：使用`~~`包裹文本，例如`~~删除线文本~~`会显示为~~删除线文本~~。
5. <u>下划线</u>：使用`<u>文本</u>`包裹，会显示为下划线。

    
# 3.列表

1. **无序列表**：使用`-`、`*`或`+`作为列表标记 。
	- ```markdown
			- 项目1
			- 项目2
				- 子项目1
				- 子项目2
	  ```

2. **有序列表**：使用数字和`.`作为列表标记。
	- ```markdown
		1. 项目1
		2. 项目2
			1. 子项目1
			2. 子项目2
	  ```

# 4.代码插入

1. **行内代码**：使用`` ` 代码` ``的形式，前后各一个`` `  ``。
2. **代码块**：使用```` ```代码``` ````的形式，并可以指定语言类型。
	- ```markdown
		\`\`\`c++
			code
		\`\`\`
	  
	  ```


# 5.引用

- 使用`>`表示引用，可以使用多个`>>`来标识多级引用。
	- ```markdown
> 		这是一条引用内容。
> 		可以有多行。
	  ```
	  
	- 显示为：
	  > 这是一条引用内容。
	  >  可以有多行。


# 6.链接
1. **行内式**：格式为`[链接文本](链接地址 "可选的悬停提示文字")`，
	- `[百度](https://www.baidu.com "这是百度的一个markdown网址")`会显示为[百度](https://www.baidu.com "这是百度的一个markdown网址")。
	- `<https://www.baidu.com>`会显示为 <https://www.baidu.com>
	- 不带`http`前缀的也可直接输入,如`www.baidu.com`,会直接显示www.baidu.com
2. **参考式**：
	- 正文重复使用`[链接文本][链接标记]`
	- 文档末尾定义`[链接标记]: 链接地址 "可选悬停提示文字"`
	- ```markdown
		  正文：
		[百度外显名称1][bd]
		[百度外显名称2][bd]
		
		末尾：
		[bd]: https://www.baidu.com "这是百度的网址"

		该方法适合一个网址多次使用，前面的外显名称有点类似C++的引用，末尾是唯一的真实链接。
	  ```

		- [百度外显名称1][bd]
		- [百度外显名称2][bd]
		
		- [bd]: https://www.baidu.com "这是百度的网址"
3. 注脚：`[^注脚内容]`
	- 文本[^这是一个注脚]

# 7.图片

1. 格式为`![替代文本](图片地址)`，会尝试显示图片，如果图片无法显示，则显示替代文本。
	-  图片地址可以是本地地址也可以是网络地址，本地地址可以使用相对路径。
	- `![C++ logo](https://example.com/cpp_logo.png)`
	- ![C++ logo](https://example.com/cpp_logo.png)
	- `![example logo](../../picture/cpplogo.png)`
	- ![example logo](../../picture/cpplogo.png)
    

  

# 8.表格

1. 使用`|`来分隔列，使用`-`来表示表头和内容的分隔线。
	- ```markdown
		| 列1 | 列2 |
		| ---- | ---- |
		| 内容1 | 内容2 |
		| 内容3 | 内容4 |
		
		想要居中则要这样
		| 列1 | 列2 |
		| :--: | :--: |
		| 内容1 | 内容2 |
		| 内容3 | 内容4 |
	  
	  ```

---

|   |   |
|---|---|
|列1|列2|
|内容1|内容2|
|内容3|内容4|

---

# 9.分割线

1. 使用`---`或`***`表示分割线,都会显示为一条分割线。
	- ```markdown
		---
		
		***
	  ```

1. ---
2. ***
